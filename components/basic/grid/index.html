    何时使用

    需要用户输入表单域内容时

    代码演示

    基础用法

    Grid 采用 flexbox 进行布局，一共分成 12 个栅格

    span=12
    span=3
    span=3
    span=3
    span=3

    Grid 可以多层嵌套使用

    span=12
    span=12
    span=6
    span=6
    span=3
    span=3
    span=3

    间距

    Grid 可以设置子元素的间距，1 个单位 = 8px。默认情况下子元素与容器存在间距。

    span=12
    span=3
    span=3
    span=3
    span=3

    间距合并后，子元素与容器元素的间距消失1。

    span=12
    span=12
    span=6
    span=6
    span=3
    span=3
    span=3

    flex 属性

    Grid 提供了大部分 flex 属性映射的配置项

    span=12
    span=3
    span=6
    span=3
