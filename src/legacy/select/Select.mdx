# Select 选择器

<Subtitle>
  下拉选择器，让用户做出选择。Select 与 Dropdown 的区别是 Select 承载的是选项，Dropdown 的承载的是操作。Select
  有三种尺寸的高度（大：40px、中：36px、小：30px），同 button。
</Subtitle>

## 参数说明

<ArgsTable of={Select} />

## 代码演示

### 默认

根据使用场景选用紧凑 padding 型的 Select，或者 固定宽度的 Select。Select 的下拉菜单最小宽度为 160px。

[Example](/?path=/story/functional-legacy-select--default)

### 多选

1. 可以通过输入及下拉选项卡进行多选。多选项以 Tag 的形式出现在输入框内。每个选项 Tag 都可以通过「X」删除。
2. Select 输入框的宽度根据页面需要设定，Select 的下拉菜单最小宽度为 160px。

[Example](/?path=/story/functional-legacy-select--multiple)

### 无边框背景的 Select

当页面的整洁度要求很高时，可以选用没有边框背景的 Select。

[Example](/?path=/story/functional-legacy-select--no-border)
