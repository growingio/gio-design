# 快速开始

## 安装

安装 `@gio-design/components` 组件包：

<TabsState initial="yarn">
  <div id="yarn" title="Yarn">

```bash
$ yarn add @gio-design/components
```

  </div>
  <div id="npm" title="NPM">

```bash
$ npm install --save @gio-design/components
```

  </div>
</TabsState>

## 使用

```javascript
import React, { ReactDOM } from 'react';
import { Button } from '@gio-design/components';

ReactDOM.render(
  <>
    <Button>GrowingIO Design</Button>
  </>,
  mountNode
);
```
