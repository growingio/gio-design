# PastTimePicker 过去时间选择框

<Subtitle>用户可以通过点击选择框，在弹出的过去时间选择器中选择相对和绝对日期区间。</Subtitle>

## 代码演示

### 常用时间

左侧 List 为常用时间和三种时间计算方式，清晰的逻辑帮助用户减少快速理解如何使用该组件。

<Canvas>
  <Story id="selectors-PastTimePicker--quick" />
</Canvas>

### 从某日至今

当用户选择“从某日至今”时，用户可以借助时间选择器选择从某一特定日期至今日的动态区间（今日日期为动态）。由于只有开始日期需要选择，我们让用户通过更改日历来做选出开始日期。

<Canvas>
  <Story id="selectors-PastTimePicker--since" />
</Canvas>

### 动态时段

当用户选择“过去 N 天”时，用户可以借助时间选择器选择过去 N 天或者过去 N~M 天，这样他可以查看（相对今天）过去一段时间范围的数据。我们让用户用过输入数字或更改日历来做出 N 天的选择。

<Canvas>
  <Story id="selectors-PastTimePicker--relative" />
</Canvas>

### 过去固定时段

当用户选择“过去固定时段”时，用户可以借助时间选择器选择日期区间的开始日期和结束日期，该日期区间为固定日期区间。

<Canvas>
  <Story id="selectors-PastTimePicker--absolute" />
</Canvas>
