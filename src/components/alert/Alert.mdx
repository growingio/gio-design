# Alert 警报（info/success/warning/error）

<Subtitle>用Alert来处理页面所有的提示吧！</Subtitle>

## 使用规范

1.常用的（info/success/warning/error）4 种颜色来处理不同类型的提示。

2.支持可关闭按钮，按钮也可绑定 onclose 触发事件。

3.可设置 icon 。
