# Loading 加载中

<Subtitle>用于页面和区块的加载中状态。</Subtitle>

## 使用规范

1. 如果 Loading 是由按钮触发的，请将 Loading 放置在按钮中，然后在 Loading 可见时禁用该按钮。
2. 如果页面的仅一部分显示新内容或正在更新，则将 Loading 放置在页面的该部分中。
3. 仅在预期等待的时间超过 1S 时显示 Loading 。
