# Empty - 空态

<Subtitle>初始化状态或无数据情况下的占位图，可以起到提示或引导操作的功能。</Subtitle>

## 应用场景

- List
- Table
- Card
- 列表页面

## 代码演示

### 未创建

用户在某一具体模块下，未创建任何内容时，可使用此类型空态。
