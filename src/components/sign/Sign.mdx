# Sign 标记

<Subtitle>图标右上角的圆形徽标数字。</Subtitle>

## 使用规范

1. 导航中的某一项目的右上角。
2. Tab 中的某一项目的右上角。
3. 头像的右上角。
4. 通知图标的右上角。

## 代码演示

### 默认-数字标记

1. 一般出现在通知图标或头像的右上角，用于显示需要处理的消息条数，通过醒目视觉形式吸引用户处理。
2. 最大显示数字标识可选「一位数」或「两位数」，「一位数」最多显示 9，超过显示 9+；「两位数」最多显示 99，超过显示 99+。
3. 当数值为「0」时，不显示标记控件。
4. 点击查看后，数字标识消失。

[Example](/?path=/story/basic-components-sign--default)

### 状态标记

1. 用不同的颜色标记状态，绿色表示正常、黄色表示警告、红色表示错误、灰色表示不可用。一般使用在列表中，如：「数据中心-事件」、「组织管理-成员管理」等。
2. 用红色小圆点表示更新状态，点击查看后，小圆点消失。
