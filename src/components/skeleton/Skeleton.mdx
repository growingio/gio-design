# Skeleton 加载中

<Subtitle>在需要等待加载内容的位置提供一个占位图形组合。</Subtitle>

1. 网络较慢，需要长时间等待加载处理的情况下。
2. 图文信息内容较多的列表/卡片中。
3. 只在第一次加载数据的时候使用。
4. 可以被 loading 完全代替，但是在可用的场景下可以比 loading 提供更好的视觉效果和用户体验。
