# Icon 图标

语义化的矢量图形。使用图标组件，你需要安装 `@gio-design/icons` 图标组件包：

```bash
npm install --save @gio-design/icons
```

或者使用 `yarn`：

```bash
yarn add @gio-design/icons
```
