# 更新日志

`GrowingIO Design` 的版本命名遵循 [日历化版本](https://calver.org/) 规范。版本格式为：Major.Minor.Micro，以下是对三个字段的解释：

- Major：主版本号使用短年号，YY - 20、21、101；
- Minor：次版本号使用月份，MM - 6、7、12；
- Micro：修订版本号是当月第几次发布，一般是每周五发布一次，有紧急 Bug 可以单独发布。
